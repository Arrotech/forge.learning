# クライアント側ファイル(Java)

Java EE Web アプリは通常、エントリ ファイル `.jsp ` を生成しますが、代わりに `.html` を生成します。このチュートリアルでは、`.html` を使用して分かりやすくします。

- `/`: `.html` (または他のプロジェクトの場合は `.jsp`)
- `/js`: `.js`
- `/css`: `.css`

これらのフォルダを `webapp` フォルダの下に作成します。下の図は、このフォルダを示しています(次のセクションでファイルを作成した後)
