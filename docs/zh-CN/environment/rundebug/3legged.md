# 在本地运行和调试

现在，您的应用程序已准备就绪，是时候运行它了。此时，我们可以测试并检查可能存在的错误（通过调试）。另外，您还可以查看提示和技巧。

## 使用示例

下一部分将介绍如何运行应用程序。在浏览器中打开它后，单击 **Autodesk Sign In** 按钮以输入您的最终用户凭据。在列表中，导航到**中心**、**项目**、**文件夹**、**项**（文件）和相应的**版本**，以在右侧查看。在树上方，显示图片和名称，以及 **refresh** 和 **sign out** 按钮。
