# 建立伺服器

您的用戶端 ID 與密碼應受到保護並保持機密性，因為您的所有檔案都將繫結至您的帳戶。對於網頁應用程式，請將其保留在伺服器上。本節將示範如何準備建立本端開發伺服器。
